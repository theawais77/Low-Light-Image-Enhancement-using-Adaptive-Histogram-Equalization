# Research-Paper-Based Low-Light Image Enhancement

Clean submission package for **Digital Image Processing Project 19**.

The project is now centered on the selected research paper:

**X. Guo, Y. Li, and H. Ling, "LIME: Low-Light Image Enhancement via Illumination Map Estimation," IEEE Transactions on Image Processing, 2017.**

## Main Submission File

Submit this Google Colab notebook:

```text
notebooks/Low_Light_Image_Enhancement_Colab_Submission.ipynb
```

The notebook is self-contained. It includes:

- Selected research paper details
- Literature review table
- LIME methodology with equations
- LIME paper implementation
- Improved LIME as innovation
- Installation cell for Colab
- Image upload cells
- Synthetic sample fallback
- Baseline methods: HE, CLAHE, Gamma+CLAHE, Retinex
- Evaluation metrics
- Histograms and CDF plots
- Visual proof of illumination maps
- Student-wise viva talking points
- ZIP download of generated results

## How to Use

1. Open Google Colab.
2. Upload `notebooks/Low_Light_Image_Enhancement_Colab_Submission.ipynb`.
3. Run cells from top to bottom.
4. Upload real low-light images when the notebook asks.
5. Download the generated result ZIP from the final cells.
6. Use the report markdown as the IEEE-style report draft.
